# plain-tower-defense

제목 : plain tower defence
Android application(java)
타워 디펜스 게임
* 기간 2020.02 ~ 2020.04

![image-play](.\tower_defense_play.png)


## 설명
* 로그인 - firebase
* 게임화면구성 - surfaceview
* 친구목록정보 - SQLite
* 상점기능 - 보석구매 페이지
* google play에 alpha 버전 등록하였으나 미승인되어 미구현
* 광고시청 후 재도전 기능 - admob
* 움직이는 버튼 - lottie animation
